Question: Describe which are the appropriate filetypes for websites.

When adding content to your website it s recommended for images to use JPEG, PNG, SVG and GIF which give you a range of compression, these four main image file types could be used as example JPEG would be used for standard images or SVG could be used so when the user enlarges it does lose its resolution and improves page speed, depending on which PNG format is mainly used for graphics like logos and have the option for a transparent background and SVG or GIF are used to animate a logo or video in small file size which makes the page run faster. For sound, MP3 and MP4 are the best file formats for web sites because all modern browsers support this format.
