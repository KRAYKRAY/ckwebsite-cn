Question: Describe how you would go about testing digital solutions.

The testing I would implement on my digital solutions would be a big range of testing example: A range of browsers across different platforms and even on different networks 3G, 4G, Wi-Fi to test stability and functionality. I Would ask someone to test as well as it is good to get someone's opinion and maybe I missed an error or layout problem.

Also, test for assessability for a range of different users so the website example is user friendly for users with a disability or no disability.If the website stores personal information i would make the security is safe and I meet all guidelines to the GDPR.
